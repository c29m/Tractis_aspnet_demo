Ejemplo de integración del servicio de identidad de tractis (https://www.tractis.com/identity) con asp.net c#.

En la carpeta \Tractis_simple se puede encontrar el código para la forma más minimalista de integración,portado 
del ejemplo de php de la documentación de tractis (https://www.tractis.com/help/?p=3537&language=es)

El resto de la aplicación utiliza los providers de membership y profiles, para crear una demo mas completa:

	- Zona privada
	- Creación de cuentas
	- Inicio de sesión con usuario/contraseña
	- Inicio de sesión con certificado vía tractis
	- Uso de iframes

Más información: http://www.dmarzo.com/2011/02/tractis-integracion-con-aspnet.html
